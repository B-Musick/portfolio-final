    CONTROLS

  • i = UP
  • k = DOWN
  • j = LEFT
  • l = RIGHT

HOW IT WORKS

- MERGE INTEGERS OF THE SAME VALUE BY PRESSING THE DIRECTION WHICH WILL MAKE THIS OCCUR.

- POINTS ACCUMULATE ACCORDING TO THE OUTCOME OF THE MERGERS (VALUE OF THE NEW BLOCK).

- NEW BLOCK WITH 2 OR 4 APPEARS AFTER EACH VALID MOVE.

- ONCE BOARD IS FULL OF VALUES AND NO MORE COMBINATIONS POSSIBLE, GAME IS OVER.
